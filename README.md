# Upgrade-System
4Story like upgrade system for wotlk.

## Installation

1. Place `Upgrade.lua` in your server's `lua` folder.
2. Unzip `Upgrade.zip` into your `addons` folder. Make sure to unzip it there.
3. Run `ItemUpgradeGenerator.sql` on your database and wait for it to finish â€” this may take a bit of time to load.
4. Run `Benson.sql` to generate the NPC with the correct model and ID for the script to hook into.
